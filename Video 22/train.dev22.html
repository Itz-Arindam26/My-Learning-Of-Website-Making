Display Property in HTML

There are below types of display property:-
  1. Inline: It takes size according to its requirement.It will not take extra space.
  2. Block: It doesn't see its requirement.It took all width of the line.In another words,It takes overspace.
We can also change the display proprty of any element by the following code in style tag... 

                <style>
                /*When element is block*/
                .anyelement{
                    display: inline;
                }

                /*When element is inline*/
                .anyelement{
                    display: block;
                }

                /*When want any element treat as partially block and partially inline*/
                .anyelement{
                    display: inline-block;
                }
                </style>
