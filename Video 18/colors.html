Lets learn about colors

Types of colors

  • Color Keyword(for example: color: aqua;)
  • Hex Color Code(for example: color: #b22222;)
  • RGB Color Code(for example: color: rgb(228, 200, 105);)
  • RGBA Color Code(for example: color: rgba(255, 149, 149, 0.8);)
  • HSL Color Code(for example: color: hsl(74, 100%, 50%);)
