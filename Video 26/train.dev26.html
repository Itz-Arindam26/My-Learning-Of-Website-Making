CSS Position Property

There are many type of position:-
  1. Static : It is default position property.
  2. Relative: It helps us to move anything anywhere easily.
  3. Fixed: It takes position according to its descendents but it where it got sticked it will not move anywhere,even after scrolling the page.
  4. Sticky: It helps us in making navigation bar.
