Types of CSS

There are 3 types of CSS given below:-
  • Inline CSS - This is a type of CSS which used directly in any targeted tag by using style attribute.
  • Internal CSS - This is used by adding <style> just after title tag.
  • External CSS - This used by attaching external CSS file by using link tag.
