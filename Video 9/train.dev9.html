Id and Classes In HTML

ID in HTML

An ID is an attribute, a unique identifier assigned to only one HTML element within a page. It is often used for unique styling and JavaScript manipulations.To target any class in CSS,It is written as ".as1004" for any class which is "as1004".It is implemented in the way in any tag given below:-

        <div id="as1001">CodeWithASamanta</div>

Classes in HTML

The class attribute lets you give the same name to multiple HTML elements. That way, you can easily change their look or behavior all at once. Classes are not unique and can be assigned to multiple elements. They are generally used for applying the same styles or behaviors to a group of elements.To target any ID in CSS,It is written as "#as1005" for any ID which is "as1005"It is implemented in the way in any tag given below:-

        <p class="as1002">CodeWithASamanta</p>
        <span class="as1002">CodeWithASamanta</span>
