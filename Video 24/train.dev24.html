Lists and its styling in CSS

Lists

We use many types of lists in any webpage.There are some list types given below:-
  1. Ordered List
  2. Unordered Lists
  3. Definition Lists

Ordered Lists

Ordered list is a type of list in which bullets are in types like 1. or i. or a. or A.These doesn't use any other bullets than the said one.
The opening tag is <ol> and closing tag is </ol>.

Unordered Lists

Unordered lists is a type of list in which bullets are like disc, square, circle etc.
The opening tag is <ul> and closing tag is </ul>.

Definition Lists

This is a type of list in which one term is define term and one of it is definition.The Opening tag is <dl> and closing tag is </dl>.
