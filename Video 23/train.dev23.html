Shadows & Outlines in HTML

Shadows By CSS

We use shadow property in CSS to give shadow to any text element as well as any box.To know how to use in style tag, see below:-

For Text Element


                    <style>
                        .anyelement{
                            text-shadow: horizental-offset vertical-offset blur color;
                        }
                    </style>

For Box element


                    <style>
                        .anyelement{
                            box-shadow: horizontal-offset vertical-offset blur spread color inset;
                        }
                    </style>

Outline by CSS

We use outline property in CSS to change style of outline.There are folllowing types of outline:-
  1. Solid
  2. Dotted
  3. Dashed
  4. Wavy
  5. Double
